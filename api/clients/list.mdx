## Endpoint

```
GET /api/v1/clientes
```

Retrieve a complete list of all registered clients in the system.

## Response

Returns an array of client objects:

<ResponseField name="[]" type="array">
  Array of client objects
  
  <ResponseField name="id" type="number">
    Unique client identifier
  </ResponseField>
  
  <ResponseField name="dni" type="string">
    National ID number (DNI) - 8 digits
  </ResponseField>
  
  <ResponseField name="nombreCompleto" type="string">
    Client's full name
  </ResponseField>
  
  <ResponseField name="correo" type="string">
    Client's email address
  </ResponseField>
  
  <ResponseField name="telefono" type="string">
    Client's phone number - 9 digits
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:8080/api/v1/clientes
```
