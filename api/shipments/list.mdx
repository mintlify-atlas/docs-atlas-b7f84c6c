## Endpoint

```
GET /api/v1/envios
```

Retrieve a list of all shipments in the system. This endpoint returns complete shipment information for administrative purposes.

## Response

Returns an array of shipment objects:

<ResponseField name="[]" type="array">
  Array of shipment objects
  
  <ResponseField name="id" type="number">
    Unique shipment identifier
  </ResponseField>
  
  <ResponseField name="codigoSeguimiento" type="string">
    Tracking code (format: ENV-{timestamp})
  </ResponseField>
  
  <ResponseField name="estado" type="string">
    Current shipment status (PENDIENTE, EN_TRANSITO, DISPONIBLE, ENTREGADO, CANCELADO)
  </ResponseField>
  
  <ResponseField name="fechaEnvio" type="string">
    Date and time when shipment was created
  </ResponseField>
  
  <ResponseField name="fechaEntrega" type="string">
    Date and time when shipment was delivered (null if not delivered)
  </ResponseField>
  
  <ResponseField name="precio" type="number">
    Shipment price
  </ResponseField>
  
  <ResponseField name="placa" type="string">
    Vehicle plate number assigned to shipment (null if not assigned)
  </ResponseField>
  
  <ResponseField name="nombreDestinatario" type="string">
    Recipient's name
  </ResponseField>
  
  <ResponseField name="dniDestinatario" type="string">
    Recipient's DNI
  </ResponseField>
  
  <ResponseField name="remitente" type="object">
    Sender information (ClienteDTO)
  </ResponseField>
  
  <ResponseField name="sucursalOrigen" type="object">
    Origin branch office (SucursalDTO)
  </ResponseField>
  
  <ResponseField name="sucursalDestino" type="object">
    Destination branch office (SucursalDTO)
  </ResponseField>
  
  <ResponseField name="encomienda" type="object">
    Package details (EncomiendaDTO)
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:8080/api/v1/envios
```
