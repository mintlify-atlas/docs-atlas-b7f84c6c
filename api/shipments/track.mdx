## Endpoint

```
GET /api/v1/envios/seguimiento/{codigo}
```

Retrieve public tracking information for a shipment using its tracking code. This endpoint returns limited information suitable for public tracking portals.

## Path Parameters

<ParamField path="codigo" type="string" required>
  The unique tracking code of the shipment (e.g., "ENV-20260309-ABCD1234")
</ParamField>

## Response

<ResponseField name="codigoSeguimiento" type="string">
  Tracking code
</ResponseField>

<ResponseField name="estado" type="string">
  Current shipment status: `PENDIENTE`, `EN_TRANSITO`, `DISPONIBLE`, `ENTREGADO`, or `CANCELADO`
</ResponseField>

<ResponseField name="fechaEnvio" type="string">
  Date and time when the shipment was created
</ResponseField>

<ResponseField name="fechaEntrega" type="string">
  Date and time when the shipment was delivered (null if not yet delivered)
</ResponseField>

<ResponseField name="precio" type="number">
  Shipment price
</ResponseField>

<ResponseField name="nombreDestinatario" type="string">
  Recipient's full name
</ResponseField>

<ResponseField name="origen" type="object">
  Origin branch office information
  
  <ResponseField name="origen.codigoSucursal" type="string">
    Branch code
  </ResponseField>
  
  <ResponseField name="origen.nombreSucursal" type="string">
    Branch name
  </ResponseField>
  
  <ResponseField name="origen.ciudad" type="string">
    City name
  </ResponseField>
</ResponseField>

<ResponseField name="destino" type="object">
  Destination branch office information
</ResponseField>

<ResponseField name="encomienda" type="object">
  Package information
  
  <ResponseField name="encomienda.peso" type="number">
    Package weight in kg
  </ResponseField>
  
  <ResponseField name="encomienda.dimensiones" type="string">
    Package dimensions
  </ResponseField>
  
  <ResponseField name="encomienda.descripcion" type="string">
    Package description
  </ResponseField>
</ResponseField>

<Note>

</Note>

## Shipment Status Meanings

- **PENDIENTE**: Shipment has been registered and is awaiting processing
- **EN_TRANSITO**: Shipment is in transit to the destination
- **DISPONIBLE**: Shipment has arrived and is available for pickup
- **ENTREGADO**: Shipment has been successfully delivered to the recipient
- **CANCELADO**: Shipment has been cancelled
