---

## Endpoint

```
GET /api/v1/envios/{id}
```

Retrieve detailed information about a specific shipment using its unique identifier.

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the shipment
</ParamField>

## Response

Returns a complete shipment object with all related information:

<ResponseField name="id" type="number">
  Unique shipment identifier
</ResponseField>

<ResponseField name="codigoSeguimiento" type="string">
  Tracking code (format: ENV-{timestamp})
</ResponseField>

<ResponseField name="estado" type="string">
  Current shipment status: PENDIENTE, EN_TRANSITO, DISPONIBLE, ENTREGADO, CANCELADO
</ResponseField>

<ResponseField name="fechaEnvio" type="string">
  ISO 8601 datetime when shipment was created
</ResponseField>

<ResponseField name="fechaEntrega" type="string">
  ISO 8601 datetime when shipment was delivered (null if not yet delivered)
</ResponseField>

<ResponseField name="precio" type="number">
  Shipment cost in local currency
</ResponseField>

<ResponseField name="placa" type="string">
  Vehicle plate number (null if not yet assigned)
</ResponseField>

<ResponseField name="nombreDestinatario" type="string">
  Full name of the recipient
</ResponseField>

<ResponseField name="dniDestinatario" type="string">
  Recipient's national ID number
</ResponseField>

<ResponseField name="remitente" type="object">
  Sender/client information
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      Client ID
    </ResponseField>
    <ResponseField name="dni" type="string">
      Client DNI
    </ResponseField>
    <ResponseField name="nombreCompleto" type="string">
      Client full name
    </ResponseField>
    <ResponseField name="correo" type="string">
      Client email
    </ResponseField>
    <ResponseField name="telefono" type="string">
      Client phone number
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="sucursalOrigen" type="object">
  Origin branch office details
</ResponseField>

<ResponseField name="sucursalDestino" type="object">
  Destination branch office details
</ResponseField>

<ResponseField name="encomienda" type="object">
  Package information
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      Package ID
    </ResponseField>
    <ResponseField name="peso" type="number">
      Weight in kilograms
    </ResponseField>
    <ResponseField name="dimensiones" type="string">
      Dimensions in format: LxWxH (cm)
    </ResponseField>
    <ResponseField name="descripcion" type="string">
      Package contents description
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:8080/api/v1/envios/1
```

## Error Responses

### Shipment Not Found

```json
{
  "error": "EnvioNotFoundException",
  "message": "Envío no encontrado con id: 999"
}
```

<Warning>
This endpoint throws `EnvioNotFoundException` if the shipment with the specified ID does not exist.
</Warning>
